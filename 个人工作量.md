### 1.2 压缩准备

杀掉所有 Node 进程 → 删除 `node_modules` 和 `.next` → 压缩 war-shape。这两个文件夹是构建产物（队友 `npm install` 会自动重建），包含在压缩包里无意义。未压缩文件夹约 254 MB（主要是 `public/data/ucdp/` 下 36 个年份的 JSON 文件，约 200 MB），压缩后 zip 约 30-50 MB。

### 1.3 沙盘文档

新增 `sandtable/3D沙盘-运行日志与使用说明.md`，包含六场战争列表、技术栈（Cesium JS + cesium-plot-js）、启动指南（需本地 HTTP 服务器）、文件清单、操作说明、数据结构、已知问题和扩展方向。

### 1.4 路径修正

`sandtable/index.html` 中 Cesium SDK 引用从 `../sdcs_3d-master/` 改为 `./sdcs/`，使沙盘文件夹可独立移动而无需外部 Cesium 安装。
